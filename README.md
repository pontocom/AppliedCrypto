# Applied Cryptography

The purpose of this project is to provide some examples that support the application of some cryptographic principles.

This project includes examples for the following:

- Classic Ciphers
- Symmetric Encryption
- Hash functions
- Message Authentication Codes
- Authenticated Encryption
- Asymmetric Encryption using RSA
- Asymmetric Encryption using DH
- Asymmetric Encryption using ECC
- Digital Signatures
- Digital Certificates
- SSL/TLS
- Post-Quantum Cryptography
- Attacks and Vulnerabilities
